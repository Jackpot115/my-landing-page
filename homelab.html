Personal Homelab

A self-hosted environment for learning and experimentation.

Overview

My homelab is a personal server setup where I experiment with self-hosting applications, network management, and cybersecurity practices. It serves as a hands-on learning platform outside of my academic studies.

Hardware & Setup

Server Hardware

  • Custom-built server with AMD Ryzen processor
  • 64GB RAM for running multiple VMs
  • 1TB NVMe SSD for fast storage
  • Multiple HDDs for data storage

Virtualization

  • Proxmox VE hypervisor
  • Multiple virtual machines
  • Container support with LXC
  • Network isolation and VLANs
Services & Applications

Productivity

  • Nextcloud for file syncing and collaboration
  • Gitea for self-hosted Git repositories
  • Wiki.js for documentation

Networking & Security

  • Pi-hole for network-wide ad blocking
  • pfSense firewall and routing
  • VPN server for secure remote access
Gaming & Entertainment

Beyond productivity, I run entertainment services like a Minecraft server for friends and family, and experiment with media streaming solutions.
